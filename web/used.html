よく使うコードテンプレ

grid

CSS 
display: grid;
grid-template-columns: 1fr 1fr 1fr 1fr;
grid-gap: 0 28px;
  • 列を均等に4分割
    grid-template-columns: repeat(4, 1fr);
  • 行を均等に3分割
    grid-template-rows: repeat(3, 1fr);
  • 同じ幅4つ
    grid-template-columns: 1fr 1fr 1fr 1fr;
  • 同じ高さ3つ
    grid-template-rows: 1fr 1fr 1fr;

flex

CSS 
display: flex;
flex-direction: column;
align-items: center;
justify-content: center;

align-item: baseline;
使用すると、 チェックボックス・ラジオボタンとテキストを揃えることができる

position

CSS 
position: absolute;
top: 50%;
left: 0;
right: 0;
transform: translate(0, -50%);
margin: 0 auto;

dlで横並び

2023.1.25
コンテンツ
html 
<dl class="dl__list">
  <dt class="dl__date">2023.1.25</dt>
  <dd class="dl__desc">コンテンツ</dd>
</dl>
css 
.dl__list {
  display: flex;
  flex-wrap: wrap; // ←不要？
  flex-direction: row;
  align-items: center;
}
.dl__date {
  width: 110px;
  font-size: 16px;
  line-height: 26px;
  letter-spacing: .03em;
}
.dl__desc {
  width: calc(100% - 110px);
  font-size: 16px;
  line-height: 30px;
}
