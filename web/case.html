CSSやjsのキャッシュが反映されない

URLの末尾にタイムスタンプを追記

CSS 
<link rel="stylesheet" type="text/css" href="http://sample.com/css/style.css?20230301">

jQueryエラー

jQuery 3.xでload()が廃止されている

JS 
// × jQuery 3.x系では動かない
$(window).load(function (){...});

// ○ jQuery 3.x系でも動く
$(window).on('load',function(){...});
参考

要素を非表示にする時

visibility: hidden
・サイズ　あり
・アニメーション　できる
・クリック　できない
※子要素で表示・非表示を上書きできる

お問い合わせのメールが届かないときの確認事項

確認メールのみ届かない

  • mailtoの設定ミス
  • 使用可能なアドレスか
  • 転送先設定確認・迷惑メールフォルダ確認してもらう

自動返信メールのみ届かない

  • メールアドレスのtype属性が "text" になってしまっている

確認メール・自動返信メール両方とも届かない

  • index.htmlがUPされている
